# diffusion-limited-aggregation

This is the process whereby particles undergo a random walk (typically referred to as brownian motion)
and when the particle reaches one that has been "stuck" it binds to it. This process usually creates these
leaf like structures, this is just a simple simulation. Run ./dla --help to see all the paramters you can tweak. 

P.S make sure to build with --release!
